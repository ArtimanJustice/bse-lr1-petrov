# Функціональні вимоги до системи LocalSquash

## Опис предметної області

LocalSquash — це локальна програмна система для стиснення медіафайлів без передавання даних на сервер. Система дозволяє користувачу обирати файл, визначати його тип, виконувати стиснення зображень, відео або аудіо, а також переглядати результат стиснення.

## Функціональні вимоги

| ID | Функціональна вимога | Пріоритет |
|---|---|---|
| FR-01 | Система має дозволяти користувачу обирати файл для стиснення. | Високий |
| FR-02 | Система має визначати тип вибраного файлу: зображення, відео або аудіо. | Високий |
| FR-03 | Система має дозволяти стискати зображення із заданим параметром якості. | Високий |
| FR-04 | Система має дозволяти стискати відео з використанням параметра CRF. | Високий |
| FR-05 | Система має дозволяти стискати аудіо із заданим бітрейтом. | Високий |
| FR-06 | Система має обчислювати розмір файлу до та після стиснення. | Середній |
| FR-07 | Система має показувати користувачу відсоток стиснення після завершення обробки. | Середній |
